# 按摩店订单管理系统

一个基于浏览器的按摩店前台订单管理系统。

## 功能特点

- 📅 技师维度甘特图，直观查看订单安排
- 📍 用户报到功能，输入手机号快速签到
- 🔔 自动完成服务，时间到自动标记已完成
- 📊 订单管理，支持大众点评/电话/微信订单
- ⚙️ 资源管理，可配置技师/床位/服务项目
- 🔄 技师轮流接单，自动推荐接单数最少的技师

## 使用方法

直接用浏览器打开 `index.html` 即可使用。

## 数据存储

数据存储在浏览器 LocalStorage 中，关闭页面不会丢失。

## 技术栈

- 纯前端 HTML/CSS/JavaScript
- 无后端依赖
- LocalStorage 数据存储
