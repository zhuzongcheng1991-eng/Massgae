# UI 设计更新说明

## 设计风格
参考 Wero UI 设计风格，采用现代简洁的卡片式设计。

## 主要变更

### 1. 布局结构
- **侧边栏导航**：新增左侧固定侧边栏，宽度 260px
- **主内容区**：右侧主内容区， margin-left: 260px
- **响应式设计**：768px 以下屏幕侧边栏变为顶部导航

### 2. 颜色系统
使用 CSS 变量定义统一的颜色系统：

```css
--primary: #667eea;       /* 主色调 */
--primary-dark: #5568d3;  /* 主色深色 */
--secondary: #764ba2;     /* 辅助色 */
--success: #10b981;       /* 成功/完成 */
--warning: #f59e0b;       /* 警告/进行中 */
--danger: #ef4444;        /* 危险/错误 */
--bg-light: #f0f2f5;      /* 浅灰背景 */
--bg-white: #ffffff;      /* 白色背景 */
--text-primary: #1f2937;  /* 主文本 */
--text-secondary: #6b7280; /* 次要文本 */
--border-light: #e5e7eb;  /* 边框色 */
```

### 3. 阴影系统
```css
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
--shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.1);
--shadow-lg: 0 10px 15px -3px rgba(0, 0, 0, 0.1);
```

### 4. 圆角系统
```css
--radius-sm: 8px;   /* 小按钮、输入框 */
--radius-md: 12px;  /* 卡片 */
--radius-lg: 16px;  /* 模态框 */
```

### 5. 组件更新

#### 侧边栏导航
- Logo 区域：显示应用名称和图标
- 导航分组：主要功能、数据、系统三个分组
- 活动状态：渐变色背景表示当前选中
- 底部信息：同步状态和日期显示

#### 顶部栏
- 页面标题
- 日期显示（右侧）

#### 统计卡片
- 4 列网格布局
- 左侧图标 + 右侧数值和标签
- 图标使用 emoji
- 数值 28px 粗体

#### 按钮
- Primary：渐变紫色背景，白色文字
- Secondary：白色背景，灰色边框
- Success/Warning/Danger：对应功能色
- Hover 效果：轻微上移 + 阴影

#### 表单元素
- 输入框：1px 边框，focus 时紫色边框 + 阴影
- 单选框：卡片式样式，选中时紫色边框
- 复选框：18px 大小

#### 模态框
- 圆角 16px
- 毛玻璃背景遮罩
- 圆形关闭按钮
- 底部浅灰色 footer

#### 订单表格
- 表头：浅灰背景，大写文字
- 行悬停效果：浅灰背景
- 状态徽章：圆角 20px pill 样式

#### 时间轴区块
- 渐变色根据订单来源区分
- 悬停效果：上移 + 阴影
- 已完成订单：灰色半透明

## 响应式断点

- **1200px 以上**：完整布局
- **768px - 1200px**：统计卡片 2 列
- **768px 以下**：侧边栏变顶部导航，单列布局

## 交互效果

### 按钮
- Hover：translateY(-1px) + 阴影
- Active：translateY(0)

### 导航项
- Hover：浅灰背景
- Active：渐变紫色背景 + 白色文字

### 输入框
- Focus：紫色边框 + 浅紫阴影

### 时间轴区块
- Hover：translateY(-2px) + 阴影

## 功能保持不变
所有业务逻辑和数据管理功能完全保持不变，仅更新视觉样式。
